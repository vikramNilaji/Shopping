# Shopping (Full-Stack MERN)

A fully functional Shopping Platform built from scratch using the MERN stack (MongoDB, Express, React, Node.js). This project implements advanced React hooks for state management and optimization alongside a secure backend API.

## 🚀 Features (Planned & In-Progress)
- **Authentication:** Secure user signup and login using JWT and bcrypt.
- **Product Management:** Browse products, search, and filter by category.
- **Shopping Cart:** Full cart functionality powered by advanced React hooks (`useReducer`, `useMemo`, `useCallback`).
- **Checkout:** Mock payment processing and order history tracking.

## 🛠️ Tech Stack
- **Backend:** Node.js, Express.js, MongoDB Atlas, Mongoose
- **Frontend:** React.js, Tailwind CSS (or chosen UI library)
